# Pour les contributeurs

Vous voulez lire, corriger ou étendre Athena. Commencez par
**[architecture](architecture.md)** et **[carte du code](code-map.md)**,
puis consultez **[développement](development.md)** pour la boucle
de développement locale.

- **[Architecture](architecture.md)** — forme du système, flux de requête.
- **[Carte du code](code-map.md)** — visite guidée du répertoire.
- **[Développement](development.md)** — installation locale, exécution
  des tests, flux des PR.
- **[Base de données](database.md)** — points saillants du schéma,
  migrations.
