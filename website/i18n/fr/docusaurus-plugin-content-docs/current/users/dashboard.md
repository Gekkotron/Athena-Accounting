# Tableau de bord

**Statut :** brouillon — contenu à venir.

## Ce que couvrira cette page

Le tableau de bord est l'endroit où Athena vous montre votre argent d'un seul coup d'œil : courbe de solde avec un axe X calendaire, donut par catégorie, panneau d'insights avec ses drapeaux d'anomalie, une vue Sankey optionnelle et un résumé des budgets. Cette page passera en revue chaque widget, les filtres par compte et par plage de dates, et les losanges des points de contrôle qui signalent une dérive par rapport à vos relevés bancaires.
