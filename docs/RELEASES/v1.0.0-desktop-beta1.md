# v1.0.0-desktop-beta1 ŌĆö first desktop beta

**The Docker ŌåÆ Tauri pivot ships.** Athena Accounting now runs as a standalone
desktop app on macOS, Linux, and Windows ŌĆö no Docker required, no server to
configure, all data local. The Docker path is unchanged for family-server
setups; this release adds a second, no-install path aimed at solo users.

Packaging is a directory-based sidecar (bundled Node runtime + native
prebuilds) rather than a single stripped binary ŌĆö the native dep tree
(sharp/libvips, @napi-rs/canvas, argon2, PGlite WASM, pdfjs, tesseract)
is hostile to single-binary bundlers, and Tauri's sidecar mechanism accepts
a folder just as happily. Bundle size is ~50ŌĆō80 MB per platform.

## macOS Gatekeeper ŌĆö "unidentified developer" workaround

The `.dmg` is **not code-signed** in this beta (no Apple Developer account
yet). On first launch macOS will show *"Athena Accounting can't be opened
because Apple cannot check it for malicious software."*

Bypass:
1. In Finder, right-click **Athena Accounting.app** and choose **Open**.
2. Confirm **Open** in the dialog that appears.
3. macOS remembers the exception; subsequent launches work normally.

See `docs/standalone-app-distribution.md` for the full walkthrough.
