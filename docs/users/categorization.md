# Categorization

**Status:** draft — content coming.

## What this page will cover

Athena categorizes transactions two ways: automatic rules that run on
every import, and manual assignment via the Tri (sorting) tab. Internal
transfers between your own accounts are detected and paired so they
don't inflate income or expense totals. This page will explain how the
rule engine matches (accent- and case-insensitive, with sign guards),
how to bulk-triage the long tail, and how retroactive re-application
preserves manual overrides.
